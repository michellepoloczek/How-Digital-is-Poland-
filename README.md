# How-Digital-is-Poland-?
This project analyses Poland's level of digitalisation in the context of the European Union using the Digital Economy and Society Index. Using Python we explore Poland's overall performance compared to:
- the EU average
- other member states
- historical trends over time.

It focuses solely on connectivity, digital skills, business technology integration, and digital public services.
Objectives:
- cleaning and preparing DESI datasets
- comparing Poland's DESI score to the EU average
- ranking Poland amongst other EU countries
- indentifying strenghts and weaknesses across DESI dimensions
- visualizing trends over time
- producing an interpretable summary of results.
